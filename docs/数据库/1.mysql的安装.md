# 安装和配置   

**安装**

一路 next 即可

**配置**

1. 用编辑器打开.bash_profile文件。
2. 加入```PATH=$PATH:/usr/local/mysql/bin```,保存并退出。
3. 执行文件：```source ~/.bash_profile```  

# 使用

命令：```mysql -u root -p```

键入安装中输入的密码即可
